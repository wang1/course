开发软件 VS Code 安装与运行

你可能了解到开发C语言程序有许多工具，但请留意，VS Code是最优秀的。

下载

请点击下载VSCode1.42便携版（集成了C/C++编译环境，便携的意思就是解压即可运行，无需正常安装）

安装

下载后的文件为Code4C.zip（240M），将其复制到非C盘, 如D盘E盘等的根目录（因权限问题，请注意不要解压到C盘）

在该文件上鼠标右键选择 全部解压缩，如下图所示（图中是把该文件复制到了E盘）。


解压后将在当前的目录下生成了一个文件夹名为Code4C，进入该文件夹，找到一个名为AutoStart.bat的文件，双击运行。 （注意：以后我们都是开发C程序都是运行该文件）

  1. 我们开发的程序都放在Projects这个文件夹里（上图红色方框），目前有一个示例的test.c的文件。点击该文件，就可在右边的编辑区显示。

如何学习

0. 左边为简明教程, 每周一次课一讲, 共15讲, 作业及实验皆在其中

1. 在线学习中心，学校各位老师的讲课视频和文档

2. C语言中文网（强烈推荐，但无需去学习需要费用的VIP部分）， 配套教材C语言小白变怪兽

3. 学习编程需要你不停的实践，这异常重要, 请牢记. Linus说过 “Talk is cheap, show me the code”
